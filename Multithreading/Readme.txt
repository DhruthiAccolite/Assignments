Assumptions:
All the passengers in the blocked queue are given the highest priority and hence assigned first.
There is only place for 5 taxis in the taxi stand.
A predefined set of 8 destinations is used to allocate to the passengers.
The taxi is allowed to wait for only 1 minute(from when it was assigned to the stand) in the stand.
The taxis and passengers are generated every 3 seconds.
For better observation passenger can be generated every 5 seconds and taxis every 8 seconds.
One thread for passengers and one thread for taxis are being used.
